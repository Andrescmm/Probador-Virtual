# Library-Flask-App

## Setup & Installtion

Make sure you have the latest version of Python and pip installed.

```
Clone the repository
```

```bash
pip install -r requirements.txt
```

## Running The App

```bash
python app.py
```

## Viewing The App

Go to `http://127.0.0.1:5000`
